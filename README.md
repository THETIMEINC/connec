## Usage

Install Node.js, then:

```shell
$ npx connec <@yourname>
```

Get the information by specifying the [connec](https://conn.ec/?ref=github) "@yourname".

### Example

https://conn.ec/@hirorock

```shell
$ npx connec @hirorock
```

You can also use github repo.

```shell
$ npx github:thetimeinc/connec @hirorock
```
